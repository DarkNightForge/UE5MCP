# OrbitForge

Agent-native tools for building and debugging real-time 3D projects.

OrbitForge explores safer, editor-native creative tooling for complex 3D scenes: structured actions, inspectable changes, and fast iteration loops.

The goal is a narrow working prototype that feels useful inside a real development workflow, with clear logs, repeatable commands, and human review before risky changes.

## Current direction

- Editor-first workflow
- Source-grounded agent actions
- Human review before risky changes
- Real-time 3D project inspection and debugging
- Clear logs, repeatable commands, and demo-friendly examples
