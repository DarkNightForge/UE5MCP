# OrbitForge project brief

## Public description

Agent-native tools for building and debugging real-time 3D projects.

## Prototype principles

1. Keep the workflow editor-native.
2. Prefer structured, reviewable actions over opaque automation.
3. Make agent activity inspectable through logs, diffs, and explicit approval points.
4. Build a narrow working proof before expanding scope.

## Event framing

OrbitForge is a creative development tool layer: it helps translate high-level plans into safe, structured actions inside complex real-time 3D projects.
